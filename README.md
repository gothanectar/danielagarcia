# Daniela Garcia - Advocacia e Assessoria Jurídica

Site profissional da advogada Daniela Garcia (OAB 108061), especializada em Direito Civil, Trabalhista, Família e Empresarial.

## 🚀 Características

- **Design Mobile-First Responsivo**: Otimizado para todos os dispositivos
- **Plantão Jurídico 24h**: Atendimento de urgência via WhatsApp
- **Interface Moderna**: Design clean e profissional
- **Performance Otimizada**: Carregamento rápido e eficiente
- **SEO Otimizado**: Meta tags e estrutura semântica

## 📱 Funcionalidades

### ✨ Principais Seções
- **Hero Section**: Apresentação principal com CTAs
- **Plantão 24h**: Box especial para urgências
- **Sobre**: Informações profissionais e credenciais
- **Serviços**: Áreas de atuação detalhadas
- **Footer**: Informações de contato e links

### 🎯 Integrações
- **WhatsApp**: Integração completa com mensagens pré-formatadas
- **Responsividade**: Mobile, Tablet e Desktop
- **Acessibilidade**: Suporte a leitores de tela e navegação por teclado

## 🛠️ Tecnologias

- **HTML5**: Estrutura semântica
- **CSS3**: Estilos modernos com Flexbox e Grid
- **JavaScript**: Interatividade e funcionalidades
- **Mobile-First**: Abordagem responsiva
- **Progressive Web App**: Service Worker para cache

## 📂 Estrutura do Projeto

